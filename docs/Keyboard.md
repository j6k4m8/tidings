# Keyboard Shortcuts

Defaults shown below. “Cmd” applies on macOS; “Ctrl” applies on Windows/Linux.

| Key          | Description     |
| ------------ | --------------- |
| <kbd>⌘</kbd> | Command (macOS) |
| <kbd>^</kbd> | Control or Ctrl |
| <kbd>⇧</kbd> | Shift           |

## Anywhere

| Action          | Description                       | Default                                  | Secondary                                |
| --------------- | --------------------------------- | ---------------------------------------- | ---------------------------------------- |
| Compose         | Start a new message.              | <kbd>⌘</kbd>/<kbd>^</kbd> + <kbd>N</kbd> | —                                        |
| Command palette | Open the command palette.         | <kbd>⌘</kbd>/<kbd>^</kbd> + <kbd>P</kbd> | —                                        |
| Open settings   | Open the settings panel.          | <kbd>⌘</kbd>/<kbd>^</kbd> + <kbd>,</kbd> | —                                        |
| Show shortcuts  | Open the keyboard shortcut sheet. | <kbd>⇧</kbd> + <kbd>/</kbd>              | <kbd>⌘</kbd>/<kbd>^</kbd> + <kbd>/</kbd> |

## Inbox / Thread list

| Action                         | Description                              | Default                                  | Secondary                                               |
| ------------------------------ | ---------------------------------------- | ---------------------------------------- | ------------------------------------------------------- |
| Focus search                   | Jump to the search field.                | <kbd>⌘</kbd>/<kbd>^</kbd> + <kbd>F</kbd> | —                                                       |
| Go to folder                   | Jump to any folder across accounts.      | <kbd>G</kbd>                             | <kbd>⌘</kbd>/<kbd>^</kbd> + <kbd>K</kbd>                |
| Go to folder (current account) | Jump to a folder in the current account. | <kbd>⇧</kbd> + <kbd>G</kbd>              | <kbd>⌘</kbd>/<kbd>^</kbd> + <kbd>⇧</kbd> + <kbd>K</kbd> |
| Toggle sidebar                 | Collapse or expand the sidebar.          | <kbd>⌘</kbd>/<kbd>^</kbd> + <kbd>\</kbd> | —                                                       |
| Next item                      | Move down the list.                      | <kbd>J</kbd>                             | <kbd>↓</kbd>                                            |
| Previous item                  | Move up the list.                        | <kbd>K</kbd>                             | <kbd>↑</kbd>                                            |
| Open thread                    | Open the selected thread.                | <kbd>⏎</kbd>                             | —                                                       |

## Thread detail

| Action    | Description                          | Default      | Secondary |
| --------- | ------------------------------------ | ------------ | --------- |
| Reply     | Reply to the selected thread.        | <kbd>R</kbd> | —         |
| Reply all | Reply to everyone in the thread.     | <kbd>A</kbd> | —         |
| Forward   | Forward the selected thread.         | <kbd>F</kbd> | —         |
| Archive   | Move the selected thread to Archive. | <kbd>E</kbd> | —         |

## Composer

| Action       | Description             | Default                                  | Secondary |
| ------------ | ----------------------- | ---------------------------------------- | --------- |
| Send message | Send the current draft. | <kbd>⌘</kbd>/<kbd>^</kbd> + <kbd>⏎</kbd> | —         |
